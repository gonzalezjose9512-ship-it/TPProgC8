# TPProgC8

Fecha: 15/04/2026

Carpeta: Ejercicio Logico

Consigna:  Almacenar 20 nombres que van a ingresar los clientes con sus respectivas edades y solo aquellos que sean mayor de 30 años podran ingresar al boliche, quienes tengan menos de 30 mostrar mensaje "A LA CASA".

Solucion: Primero definiremos una cantidad fija de 20 clientes para luego crear dos array uno para guardar los nombres y otro para las edades mediante un ciclo for, solicita al usuario ingresar el nombre y edad de cada persona. Una vez cargados los datos sera evaluado por el sistema. Si la persona tiene más de 30 años el sistema indica que puede ingresar caso contrario, muestra el mensaje "a la casa".
